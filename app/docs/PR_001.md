# PR: Version Compare & Evolution Visualization (InnerMirror Core)

## 🎯 핵심

InnerMirror의 핵심 개념인  
👉 **“판단의 변화와 진화”를 시각화하는 기능을 최초로 구현**

---

## ✨ 주요 기능

- 판단 기록을 **버전 단위(V1, V2, …)** 로 저장
- 이전 버전과 비교하여 **변화 신호(diff) 생성**
- 변경된 항목을 **UI에서 직접 비교 표시**
- 판단 흐름을 보여주는 **타임라인 구조 추가**
- **이전 버전 불러오기 기능**으로 사고 연속성 확보

---

## 🧠 의미

기존:
- 단순 기록 시스템

이번 변경:
- 👉 **판단 진화 추적 시스템으로 전환**

---

## 🔥 변화 구조
```
기록 → 버전 → 비교 → 변화 신호 → 진화
```

---

## 📌 기대 효과

- 사용자가 자신의 사고 변화를 인식 가능
- 반복 학습이 “기록”이 아니라 “진화”로 전환
- InnerMirror 엔진의 기반 데이터 구조 확보

---

## 🧪 테스트

1. 판단 기록 입력
2. 여러 번 수정하여 저장
3. 이전 버전과 변화 항목 확인
4. 타임라인에서 흐름 확인
