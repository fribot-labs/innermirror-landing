# Landing Runtime Mapping Layer

This folder belongs to the Landing repository.

Despite the folder name, this is **not** the private Runtime.

This folder contains Runtime response mapping utilities.

Its purpose is to transform structured Runtime responses into UI-friendly presentation data for Landing components.

---

## Allowed Responsibilities

This folder may contain:

- Runtime response mapping
- presentation data shaping
- surface data creation
- timeline item mapping
- fallback display data
- UI-friendly labels derived from Runtime responses

---

## Not Allowed

This folder must not contain:

- Reflection analysis
- Summary generation
- Runtime question generation
- Decision Review generation
- PBL Coaching generation
- Continuity Intelligence
- Runtime Memory processing
- Runtime orchestration
- proprietary AI reasoning

---

## Boundary Rule

Runtime generates intelligence.

Landing maps Runtime intelligence into presentation data.

This folder is a presentation mapping layer, not a Runtime intelligence layer.
