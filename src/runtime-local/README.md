# Runtime Local UI Cache

This folder belongs to the Landing repository.

It contains temporary client-side persistence used for user experience continuity.

---

## Purpose

This folder supports local UI recovery and temporary Reflection persistence in the browser.

It is not Runtime Memory.

---

## Allowed Responsibilities

This folder may contain:

- temporary local Reflection persistence
- offline recovery support
- local UI cache refresh
- local cache clearing
- client-side draft continuity

---

## Not Allowed

This folder must not contain:

- long-term Runtime Memory
- Memory Intelligence
- Reflection analysis
- Decision Review generation
- PBL Coaching generation
- Continuity Intelligence
- Runtime orchestration
- proprietary AI reasoning

---

## Boundary Rule

Local UI cache is not Runtime Memory.

Runtime Memory belongs to `innermirror-runtime-private`.

This folder exists only to protect the user experience during temporary local or network interruptions.
