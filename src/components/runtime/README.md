# Runtime Presentation Components

This folder belongs to the Landing repository.

It contains React components that present Runtime-derived results to the learner.

---

## Purpose

These components display structured Runtime responses.

They do not generate Runtime intelligence.

---

## Allowed Responsibilities

This folder may contain components for:

- Runtime status display
- Reflection continuity display
- Runtime memory timeline display
- recovery notice display
- offline sync notice display
- coaching result display
- decision review display

---

## Not Allowed

This folder must not contain:

- Reflection analysis
- Summary generation
- Runtime question generation
- Decision Review generation
- PBL Coaching generation
- Continuity Intelligence
- Runtime Memory processing
- Runtime orchestration
- proprietary AI reasoning

---

## Boundary Rule

Components display Runtime results.

They do not create Runtime results.

Presentation belongs to Landing.

Intelligence belongs to Runtime.
