# InnerMirror — Decision Evolution System

## 🎯 Problem

AI makes knowledge accessible to everyone.

But people still struggle with:
- making decisions
- improving their thinking
- understanding how their judgment changes

---

## 💡 Solution

InnerMirror helps users:

👉 record decisions  
👉 track changes over time  
👉 visualize thinking evolution  

---

## ⚙️ How it works
```
Think → Record → Version → Compare → Evolve
```

Users:
1. Define a problem
2. Record their decision
3. Create new versions
4. Compare changes
5. Learn from evolution

---

## 🔥 Differentiation

| Existing Tools | InnerMirror |
|------|--------|
| Notes | Decision evolution |
| Static | Version-based |
| Memory | Change tracking |

---

## 📊 Core Concept
```
ΔV = Range + Frequency + Impact
```

Decision change becomes **data**.
